# Gushwork Assignment

A responsive landing page assignment built as a static frontend project using HTML, CSS, and vanilla JavaScript.

## Live Demo

- Production: <https://abishekmahi.github.io/gushwork/>
- Portfolio: <https://abishekmahi.netlify.app/>

## Project Overview

This project showcases a polished product marketing page with:

- responsive layout behavior across desktop, tablet, and mobile breakpoints
- interactive hero gallery and section carousels
- manufacturing process step navigation
- FAQ accordion behavior
- modal interactions and smooth section navigation
- resource download cards and contact CTA section

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES modules)

## Running Locally

Since this is a static frontend project, you can run it in either of these ways:

1. Open `index.html` directly in a browser.
2. Use a local development server such as VS Code Live Server for a better development experience.

## Notes

- The styling is organized into modular CSS files for easier maintenance.
- JavaScript behavior is split into focused modules and initialized through `js/main.js`.
- The layout has been tuned for multiple breakpoints with special mobile-specific UI adjustments.
